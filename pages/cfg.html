MxM Context Free Grammar Assignment by Dor Rondel

Context Free Grammar

Intro

I have debatedily created a context free grammer that creates good music. It does so by abiding with the dominant notes in the C, E, and Dorian scales. Additionally, it allows for more flexibility by having mappings that may alter the tempo of the resulting meoldy. In this assignment, I came to appreciate the advantages context free grammars, have over l-systems. Whereas in L-systems, you have potentially infinite repetitions of a subset ordered notes, width CFGs, you have more variety, and running the same notes through the mappings can produce different outputs each time, as each nonterminal symbol can map to various nonterminals (unlike in L-systems).

This is the context free grammar that was devised for the assignment

                    Terminals { PLAY, 2PLAY, RP, LP, 2RP, 2LP, 3RP, 3LP, 4RP, 4LP, 2X2, 2S2 }

                    Nonterminals: { seq, fast, slow, cmaj, emin, dor }
                    
                    Starting symbol: seq
                    
                    Productions:

                    seq >> slow fast
                    seq >> slow fast slow fast
                    seq >> slow slow fast fast 
                    seq >> fast fast slow slow
                    seq >> fast slow fast slow
                    slow >> 2s2 cmaj 2x2
                    slow >> 2s2 emin 2x2
                    slow >> 2s2 dor 2x2
                    fast >> 2x2 cmaj 2s2
                    fast >> 2x2 emin 2s2
                    fast >> 2x2 dor 2s2
                    cmaj >> 2play 2rp play 2rp play 4lp
                    emin >> 2rp 2play lp play 3lp play 2rp
                    dor >>  2lp 2play 3rp play 2rp play 3lp
